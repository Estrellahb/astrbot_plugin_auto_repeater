# auto_repeater

AstrBot 插件 —— 关键字回复 + 复读机

- 支持多关键词匹配（一个资源配多个触发词）
- 精确匹配 + difflib 模糊匹配兜底
- 同一句话只复读一次
- 所有配置通过后台设置面板管理

---

## 配置项

| 配置项 | 类型 | 说明 |
|--------|------|------|
| 开启复读机 | 开关 | 连续 N 个不同人说相同内容时复读 |
| 复读触发人数 | 滑块 2-10 | N 个不同的人连续说相同内容时触发复读 |
| 开启关键字回复 | 开关 | 命中关键词自动回复 |
| 模糊匹配阈值 | 滑块 0.3-1.0 | 精确没命中时启用模糊匹配，越高越严格 |
| 关键词列表 | 文本框 | JSON 格式，每条含 keywords 和 reply |

---

## 关键词格式

```json
{
  "资源名": {
    "keywords": ["触发词1", "触发词2"],
    "reply": "回复内容"
  }
}
```

示例：

```json
{
  "kazumi": {
    "keywords": ["看番", "樱花动漫", "动漫", "追番", "番剧软件"],
    "reply": "kazumi，下载：https://kazumi.app"
  },
  "mihon": {
    "keywords": ["漫画", "tachiyomi", "阅读器", "看漫画"],
    "reply": "Mihon，下载：https://mihon.app"
  },
  "早安": {
    "keywords": ["早", "早上好", "早安", "起床"],
    "reply": "早上好呀~"
  }
}
```
